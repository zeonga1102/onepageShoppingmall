# onepageShoppingmall

다만 컨셉이 책 리뷰가 아닌 상품의 주문과 조회이며 소정의 기능들을 추가했습니다.
1. api를 이용해 실시간으로 달러-원 환율을 보여줍니다.
2. meta 태그를 추가했습니다. (og:title, og:image, og:description)<br>
